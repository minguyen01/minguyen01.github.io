Atlantic Search Engine

Feb. 2022 - Mar. 2022

This is a search engine that I created in an information retrieval course from the ground-up using Python, HTML, and CSS. The backend component uses BeautifulSoup and PorterStemmer to parse webpages. The frontend component uses Flask. The search engine's current capabilities includes: indexing a very large set of webpages given to it, accepting queries of varying length in order to return relevant results, and providing a clear U.I. for users to search without any complications. One-term queries usually run around 10-30 ms while very long queries can range from 200-300 ms.

Swyper

Nov. 2019

This is a web application that I and three other students worked on together at a beginner-oriented hackathon. My role in this project is frontend designer. We were given several hours for two days in order to work on it. We were able to get the 2nd most voted project by the other contestants. Swyper is meant to be a centralized platform for students at UCI to be able to sell "swipes" to one another. Swipes can be thought of as passes to the cafeterias at UCI, in which there was no concrete way of selling them to one another besides posting on the UCI's Facebook pages and hoping that someone would take the offer.

Aspect Alchemy

Oct. 2019 - Dec. 2019

This is a game that I and five other students worked on together for the Video Games Development Club at UCI. My role in this project is game designer, where I give ideas on concepts for the game along with playtesting it and in order to report on areas that the game can be adjusted on. Aspect Alchemy is a game developed in Unity. In the game, you hire workers to send out to various areas in order to collect resources. At the end of the day, you can use the resources you gathered to craft new items. The end goal is to craft an item called the "Philosopher's Stone."
